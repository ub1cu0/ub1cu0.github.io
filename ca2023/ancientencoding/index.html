Ancient Encoding



Enumeración


Al descomprimir el zip nos quedan 2 archivos, un output.txt y un source.py

❯ ls
 output.txt   source.py

Al leer el archivo output.txt nos encontramos una cadena probablemente cifrada

❯ cat output.txt
0x53465243657a467558336b7764584a66616a4231636d347a655639354d48566664326b786246397a5a544e66644767784e56396c626d4d775a4446755a334e665a58597a636e6c33614756794d33303d

En el script de python podemos ver como cifra la flag, primero le hace un b64encode seguido de un hex a su resultado para guardarlo en el output.txt

❯ cat source.py
from Crypto.Util.number import bytes_to_long
from base64 import b64encode

FLAG = b"HTB{??????????}"


def encode(message):
    return hex(bytes_to_long(b64encode(message)))  


def main():
    encoded_flag = encode(FLAG)
    with open("output.txt", "w") as f:
        f.write(encoded_flag)


if __name__ == "__main__":
    main()


Explotación


Podemos hacer el proceso inverso decodeando la cadena de hexadecimal a ascii y luego de base64, de esta manera la decodeamos y podemos leer la flag

❯ xxd -ps -r output.txt | base64 -d
HTB{1n_y0ur_j0urn3y_y0u_wi1l_se3_th15_enc0d1ngs_ev3rywher3}
